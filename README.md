## How to Play

Visit https://audio-geoguessr.vercel.app/ to play

## About

This web app uses field recordings from aporee.org, who self-describe their project as *cartography of sounds from all around the world*. They have over a quarter-century worth of user-uploaded, high-fidelity audio recordings. 

Each sound recording is protected under a unique Creative Commons license from the original creator. This project does not change, manipulate, adapt, or build upon the recordings in any way. This web app is not a commercial product.
